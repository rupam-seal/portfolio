## ⚗️ Installation

Clone the project

```bash
  git clone https://github.com/rupam-seal/portfolio-2022.git
```

<br>

Go to the project directory

```bash
  cd portfolio-2022

```

<br>

Install Dependencies

```bash
  npm install

```

<br>

Run server

```bash
  npm start
```
